# About
Quilt is a programming language made by me.

# Downloading
## Linux/Mac
- Download `launcher.sh`
- Run `bash launcher.sh`

## Windows
- Download [Python](https://python.org)
- Download `launcher.bat`
- Run `launcher.bat`
